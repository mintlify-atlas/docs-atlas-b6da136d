## Action Menu Shortcuts

The Action Menu is your central hub for accessing Tabby AI features. Use these shortcuts to quickly open and navigate the menu.

## Opening the Action Menu

| Shortcut | Action |
| --- | --- |
| `Ctrl+\` | Open/close action menu |

<Tip>
  The action menu provides quick access to all major Tabby AI features. Press `Ctrl+\` from anywhere to open it.
</Tip>

## Navigation Within Menus

Once the action menu or any panel is open, use these shortcuts to navigate:

| Shortcut | Action |
| --- | --- |
| `Esc` | Close menu or go back |
| `Enter` | Select current item / Accept result |
| Arrow keys | Navigate menu items |

## Related Global Actions

These shortcuts provide quick access to specific features without opening the action menu:

| Shortcut | Action | Description |
| --- | --- | --- |
| `Ctrl+Space` | Get AI suggestion | Instantly get AI suggestions without opening menu |
| `Ctrl+Shift+B` | Toggle brain panel | Open/close the brain panel |
| `Ctrl+Alt+I` | Interview ghost text | Enable ghost text for interview mode |
| `Ctrl+Alt+J` | Voice Agent | Launch the voice agent |

<Note>
  The action menu remembers your most recently used features for faster access.
</Note>

## Window Management

When working with floating panels and windows:

| Shortcut | Context | Action |
| --- | --- | --- |
| `Ctrl+Arrow` | Any floating window | Move window in the direction of arrow key |
| `Esc` | Any panel | Close panel |

<Tip>
  Use `Ctrl+Arrow` keys to reposition floating windows without using your mouse, keeping your hands on the keyboard.
</Tip>
