## Interview Copilot Shortcuts

The Interview Copilot is designed to help you during coding interviews and problem-solving sessions. These shortcuts give you quick access to AI-powered analysis and suggestions.

<Note>
  Interview Copilot shortcuts are only active when the copilot feature is enabled.
</Note>

## Core Copilot Actions

| Shortcut | Action | Description |
| --- | --- | --- |
| `Alt+X` | Analyze coding problem | AI analyzes the current coding problem and provides insights |
| `Alt+Shift+X` | Update with new constraints | Update the analysis with new problem constraints or requirements |
| `Alt+N` | Code suggestions | Get AI-powered code suggestions for the current context |

## Tab Navigation

| Shortcut | Action |
| --- | --- |
| `Ctrl+1` | Switch to tab 1 |
| `Ctrl+2` | Switch to tab 2 |
| `Ctrl+3` | Switch to tab 3 |
| `Ctrl+4` | Switch to tab 4 |
| `Ctrl+5` | Switch to tab 5 |
| `Ctrl+6` | Switch to tab 6 |

<Tip>
  Use tab switching shortcuts to quickly navigate between different views or problem scenarios during an interview.
</Tip>

## Ghost Text & Transcription

These global shortcuts are particularly useful during interviews:

| Shortcut | Action | Description |
| --- | --- | --- |
| `Ctrl+Alt+I` | Interview ghost text | Toggle ghost text suggestions while you type |
| `Ctrl+Shift+T` | Cycle transcribe modes | Switch between different transcription modes |
| `Ctrl+Alt+T` | Toggle voice transcription | Enable/disable voice-to-text transcription |

## Common Actions

| Shortcut | Context | Action |
| --- | --- | --- |
| `Enter` | Result panel | Accept suggestion and paste into editor |
| `Esc` | Any panel | Close panel or go back |
| `Ctrl+Space` | Editor | Get AI suggestion for current context |

<Note>
  During live interviews, consider using voice transcription (`Ctrl+Alt+T`) to verbalize your thinking while coding.
</Note>
