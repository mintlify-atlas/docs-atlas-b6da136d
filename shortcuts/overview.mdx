## All Keyboard Shortcuts

Tabby AI Keyboard provides a comprehensive set of keyboard shortcuts to enhance your productivity. This page lists all available shortcuts organized by context.

<Note>
  Most shortcuts use `Ctrl` as the modifier key. On macOS, you may need to use `Cmd` instead depending on your system settings.
</Note>

## Global Shortcuts

These shortcuts work from anywhere in the application:

| Shortcut | Action |
| --- | --- |
| `Ctrl+\` | Open/close action menu |
| `Ctrl+Space` | Get AI suggestion |
| `Ctrl+Shift+B` | Toggle brain panel |
| `Ctrl+Alt+I` | Interview ghost text |
| `Ctrl+Alt+J` | Voice Agent |
| `Ctrl+Shift+X` | Stop Autotyping |
| `Ctrl+Shift+T` | Cycle Transcribe modes |
| `Ctrl+Alt+T` | Toggle voice transcription |

## Copilot Shortcuts

These shortcuts are specific to the Interview Copilot feature:

| Shortcut | Action |
| --- | --- |
| `Alt+X` | Analyze coding problem |
| `Alt+Shift+X` | Update with new constraints |
| `Alt+N` | Code suggestions |
| `Ctrl+1-6` | Switch tabs |

## Navigation Shortcuts

These shortcuts help you navigate and interact with the UI:

| Shortcut | Context | Action |
| --- | --- | --- |
| `Ctrl+Arrow` | Any window | Move floating window |
| `Esc` | Any panel | Back/close |
| `Enter` | Result | Accept & paste |

<Tip>
  You can customize many of these shortcuts in the settings panel to match your preferred workflow.
</Tip>
