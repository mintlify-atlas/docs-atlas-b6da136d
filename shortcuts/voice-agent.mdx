## Voice Agent Shortcuts

The Voice Agent enables hands-free interaction with Tabby AI. Use these shortcuts to control voice input, transcription, and voice-activated features.

## Voice Agent Control

| Shortcut | Action | Description |
| --- | --- | --- |
| `Ctrl+Alt+J` | Launch Voice Agent | Open and activate the voice agent |
| `Ctrl+Alt+T` | Toggle voice transcription | Enable or disable voice-to-text transcription |
| `Ctrl+Shift+T` | Cycle transcribe modes | Switch between different transcription modes |

<Tip>
  Use `Ctrl+Alt+J` to quickly activate voice commands when you need hands-free operation.
</Tip>

## Transcription Modes

Tabby AI offers multiple transcription modes for different use cases:

<Tabs>
  <Tab title="Continuous">
    Continuously transcribes your voice input without manual activation. Best for extended dictation or explaining code logic.
  </Tab>
  <Tab title="Push-to-Talk">
    Transcribes only while a key is held down. Provides precise control over when voice input is captured.
  </Tab>
  <Tab title="Voice-Activated">
    Automatically detects when you're speaking and starts transcription. Balances convenience with accuracy.
  </Tab>
</Tabs>

<Note>
  Press `Ctrl+Shift+T` to cycle through these modes and find the one that works best for your workflow.
</Note>

## Autotyping Control

When the Voice Agent is actively typing text based on your voice input:

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+X` | Stop autotyping | Immediately stop the voice agent from typing |
| `Esc` | Cancel/close | Close the voice agent panel |

<Tip>
  If the voice agent starts typing something unexpected, quickly press `Ctrl+Shift+X` to stop it, then `Ctrl+Z` to undo if needed.
</Tip>

## Voice + Interview Copilot

Combine voice features with Interview Copilot for a powerful interview experience:

| Shortcut | Action | Use Case |
| --- | --- | --- |
| `Ctrl+Alt+I` | Interview ghost text | Enable AI suggestions while speaking |
| `Alt+X` | Analyze coding problem | Verbally describe problem, then analyze |
| `Alt+N` | Code suggestions | Get voice-activated code suggestions |

## Common Actions

| Shortcut | Context | Action |
| --- | --- | --- |
| `Enter` | Transcription result | Accept and paste transcribed text |
| `Esc` | Voice panel | Close voice agent panel |
| `Ctrl+Space` | Any context | Get AI suggestion (can be voice-triggered) |

<Note>
  Voice Agent features require microphone permissions. Make sure to grant access when prompted.
</Note>

## Best Practices

- **Clear Speech**: Speak clearly and at a moderate pace for best transcription accuracy
- **Noise Reduction**: Use in a quiet environment or with a quality microphone
- **Mode Selection**: Choose the transcription mode that matches your workflow
- **Emergency Stop**: Remember `Ctrl+Shift+X` to quickly stop unwanted autotyping
